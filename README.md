# Connect Four
A classic Connect Four game implemented in C, featuring both local multiplayer and an AI mode.

## Project Structure
- **src/**: Contains all `.c` source files.
- **include/**: Contains `.h` header files for function prototypes and definitions.
- **database/**: Stores game logs and player information.

## Getting Started
### Prerequisites
- A C compiler (like `gcc`)
- `make` build tool

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ange-TOSSOU/Connect_Four.git
   cd Connect_Four
   ```
2. Build the project
   ```bash
   make
   ```

### How to Play ?
1. Run the compiled executable:
   ```bash
   ./connect4
   ```
2. Enjoy the game:
   - Play against a friend locally.
   - Play against the computer.
   - Watch a previously saved match.
   - Display statistics on previous games.
   - Toggle game parameters.
   - Look at the quick guide on the rules and the specific keyboard controls.
